<h1 align="center" id="sobre">🔴 ➕ Alura Combo+ ➕ 🔴</h1>

<p>
  Projeto final do curso <a href="https://cursos.alura.com.br/course/html-css-praticando-html-css">HTML e CSS: praticando HTML/CSS</a>.
</p>

</p>
